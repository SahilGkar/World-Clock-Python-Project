World Clock (Python + Tkinter)

This is a simple world clock made using Python and Tkinter.
It shows the current time in different cities around the world, updates automatically every second, and lets you switch between 12-hour and 24-hour formats.
Your local timezone is detected and highlighted automatically.

Features:
- Real-time clock updates
- Shows major world cities
- 12H / 24H format toggle
- Local timezone highlight
- Gradient background for a clean look
- Lightweight and easy to run

Libraries used:
- tkinter
- pytz
- datetime
- tzlocal

How to run:
1. Make sure Python 3 is installed.
2. Install the required libraries:
   pip install pytz tzlocal
3. Run the script:
   python world_clock.py
